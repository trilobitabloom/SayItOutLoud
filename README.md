# 吾聲的吶喊 - Say It Out Loud

《吾聲的吶喊 Say it out loud》是一款以職場性別議題為背景的懸疑推理文字冒險遊戲。  

玩家將以第一人稱視角扮演行銷部門的菜鳥員工〈吳聲〉，隨著吳聲日復一日的工作，開始察覺到一絲不對勁，同事間尖銳的詞彙、意有所指的總是朝向公司裡的「某個人」。
同事口中成天不斷談論的「那個人」，究竟是何方神聖？
而公司裡似乎還有個所有人都心知肚明、卻又不敢公開談論的「秘密」.......
